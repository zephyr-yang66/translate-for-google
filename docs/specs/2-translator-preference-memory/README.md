# 功能 2：翻译工具偏好记忆

## 概述

这个功能让翻译插件能够记住用户上次使用的翻译工具，并在用户切换工具时提供智能的配置验证和引导。

## 核心价值

1. **提升用户效率**：用户无需每次都重新选择翻译工具
2. **减少配置错误**：主动检测配置缺失，引导用户完成配置
3. **改善用户体验**：提供清晰的保存反馈和错误提示

## 用户故事概览

### P1 - 记住上次使用的翻译工具
用户选择翻译工具后，系统自动记住，下次使用时无需重新选择。

### P1 - 配置验证与引导
切换工具时检测配置完整性，如果缺少配置则引导用户前往配置页面。

### P2 - 配置保存验证与反馈
保存配置时验证完整性，提供清晰的成功/失败反馈。
